# 皮影·灵韵项目前端网页介绍

## 前端核心环境依赖介绍
| 核心环境/依赖版本               | 原因 |
|-------------------------| ----  |
| Vue2.7.16               | 选择 2.7+。它是 Vue 2 的最终形态，内置了 Composition API，且对 Vite 的支持是官方原生的（无需额外转换插件），可以兼容 React 现代生态 |
| Vite6.x                 | 与 React 项目中的 rolldown-vite 保持同代，能同时驱动 Vue 和 React 两个引擎 |
| Node.js20.x             | React 19 和 Vite 6 需要现代 Node 环境。|
| @vitejs/plugin-vue2     | 官方维护的 Vue 2 专用 Vite 插件，性能最稳定。|
| @vitejs/plugin-react    | 用于解析 React 的 JSX 语法，确保 React 小项目能直接作为源码引入。|
